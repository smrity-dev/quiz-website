# 🎯 Quiz Website

A modern and responsive Quiz Website built using **HTML5, CSS3, and JavaScript**. This project allows users to answer multiple-choice questions, calculate scores instantly, and restart the quiz after completion.

---

## 📌 Project Overview

The Quiz Website is a frontend web application developed using core web technologies. It is designed to provide an interactive quiz experience with a clean user interface and responsive layout.

This project demonstrates JavaScript DOM manipulation, event handling, conditional logic, and responsive web design.

## 💻 How It Works

1. User Selects Department and Year
2. User starts the quiz button.
3. Questions are displayed one by one.
4. User selects an answer.
5. JavaScript checks the selected option.
6. Score is updated.
7. Final result is displayed.
8. User can restart the quiz.
